# Stubs de Laravel AI SDK

Estos archivos (`.stub`) se publicaron para personalizar la salida de `php artisan make:agent`, `make:tool` y `make:agent-middleware`.

Si no los editáis, podéis borrar esta carpeta y usar los stubs por defecto del paquete; si los mantenéis versionados, actualizadlos cuando actualicéis `laravel/ai` para no divergir del SDK.
